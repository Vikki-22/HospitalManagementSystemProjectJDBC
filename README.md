 HMS - Hospital Management System 🏥

A robust and modular **Hospital Management System** developed using **Java (JDK 21), JDBC, and MySQL**.
This project follows the **MVC (Model-View-Controller)** architecture to manage hospital workflows like doctor scheduling, patient registrations, appointments, billing, doctor reports, and patient history using a clean console-based interface.

---

 ✨ Features

 🏢 Department Management

* Add new hospital departments
* Update department details
* Manage department locations
* Track department information

---

 👨‍⚕️ Doctor Management

* Add doctor profiles
* Manage doctor specializations
* Store doctor contact details
* Allocate doctors to departments
* Fetch doctor details using Doctor ID

---

 🧑 Patient Management

* Register new patients
* Store blood groups and patient details
* Manage patient contact information
* Maintain patient records

---

 📅 Appointment Scheduling

* Book appointments
* Cancel appointments
* Complete appointments
* Link patients with doctors
* Manage appointment dates and visit reasons

---

 📜 Patient Appointment History

* View complete patient appointment history
* Display previous appointments
* Show doctor consultation details
* View appointment reasons

---

 📊 Doctor Appointment Report

* Generate doctor-wise appointment reports
* Track total appointments handled by doctors
* View doctor specialization reports

---

 💳 Billing System

* Generate patient invoices
* Track billing amounts
* Manage payment statuses
* Handle pending and paid bills

---

---

 📁 Project Structure

 🔹 Controller
Handles application logic and user requests.

 🔹 Entity
POJO classes representing database tables.

 🔹 ModelDao
Contains database operations and connection utilities.

 🔹 View
Handles presentation and user interaction.

---

 🗄️ Database Functionalities

* Insert Records
* Update Records
* Delete Records
* Read Single Record
* Read Multiple Records
* JOIN Queries


---

 🚀 Run Project

 1️⃣ Configure Database

Run the `database.sql` file in MySQL Server to create tables and dummy data.

 2️⃣ Add JDBC Driver

Add MySQL JDBC Connector Jar file to the project libraries.

 3️⃣ Run Application

Execute `MainController.java` to start the project.

 🚀 MySQL Schema and Connectivity
